# gitignore
### 과정
1. .gitignore 파일 생성
   ```bash
   touch .gitignore
   ```
2. a.txt, b.txt 파일 생성
3. gitignore 에 a.txt 작성
4. git init
5. git status

<br>

>[!WARNING]
>이미 git의 관리를 받은 이력이 있는 파일이나 디렉토리는 나중에 gitignore에 작성해도 적용되지 않는다.
>```bash
>git rm --cached
>```
>명령어를 통해 git 캐시에서 삭제가 필요하다.
