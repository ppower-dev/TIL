선택자(selector)

선택자의 개념

  • 무엇을 꾸민다는 것은, 일단 '선택(select)'해야 가능한 거 아니겠어? 그래서 CSS 에선 class값으로 '이름표'를 달아줘. 예를 들어, mytitle이라고 이름표를 달아준 뒤, "mytitle 이름표가 붙은 부분들의 글씨를 빨간색으로 바꿔라"라고 하는거지.
  • 잠깐! 선택자(selector) 이름 잘 짓기!
    선택자 이름은 자유롭게 지을 수 있어. 하지만, 다른 사람들과 협업할 때는 이름을 잘 지어야 해.

그럼 CSS는 어떻게 사용하냐???

  • <head> ~ </head> 안에 <style> ~ <style/> 로 공간을 만들어서 작성해.
  • 이 코드의 head 태그 안의 style 태그 안에서 red-font 라는 클래스를 잘 보자. 단, red-font라는 클래스를 가리킬 때, .red-font{...}라고 써줘야 하는 것을 꼭! 기억하자! 이름 앞에 .을 빼먹지 마!!!
